## Dev Notes

### Refator do `getMessageOptions`
Antes (formato Baileys, retornado pra `wbot.sendMessage`):
```ts
return { image: fs.readFileSync(path), caption: body, mimetype };
```

Depois (formato uazapi):
```ts
return { type: "image", file: base64OrUrl, text: body, mimetype };
```

Onde `file` pode ser:
- URL HTTPS pĂşblica (preferido pra mĂ­dias servidas pelo nosso backend)
- Base64 inline (fallback pra mĂ­dias locais que nĂŁo tĂŞm URL pĂşblica)

### PseudocĂłdigo loop de campanha
```ts
for (const shipping of campaignShippings) {
  try {
    const options = await getMessageOptions(file.path, ...);
    if (options.type === "text") {
      await SendText(whatsapp, { number: chatId, text: options.text });
    } else {
      await SendMedia(whatsapp, { number: chatId, ...options });
    }
  } catch (err) {
    if (err.code === "ERR_UAZAPI_RATE_LIMITED") {
      // re-enqueue com delay maior
      await campaignQueue.add(...);
      break;
    }
    throw err;
  }
  await delay(campaign.delay);
}
```
