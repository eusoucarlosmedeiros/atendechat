# Story 01.06a — Campanhas (queues.ts) + helpers/SendMessage via uazapi

## Status
**Draft** — depende de 01.02 Done.

## Story
**As a** sistema de campanhas
**I want** que `queues.ts` e `helpers/SendMessage.ts` enviem via uazapi em vez de Baileys
**So that** disparos em massa funcionem após o cutover.

## Acceptance Criteria
1. Em `backend/src/queues.ts`, todas as 3 chamadas a `wbot.sendMessage(chatId, ...)` (linhas ~692, 710, 717) são substituídas por `SendText(whatsapp, ...)` ou `SendMedia(whatsapp, ...)`.
2. `helpers/SendMessage.ts`: substituir `wbot.sendMessage(chatId, { text })` por `SendText`. Se houver `mediaPath`, usar `SendMedia`.
3. `services/WbotServices/SendWhatsAppMedia.ts:getMessageOptions` é refatorado para retornar **payload no formato uazapi** (`{ type, file, text, docName }`) em vez do formato Baileys (`{ image, caption, mimetype }`). Todos os callsites (incluindo `queues.ts`) passam a usar o formato novo.
4. **Rate limit**: campanha grande pode hit 429. O loop de envio em `queues.ts` precisa respeitar o `delay` configurado (campanha) E re-enfileirar via Bull em caso de `ERR_UAZAPI_RATE_LIMITED` propagado pelo cliente.
5. Bull continua sendo a engine de scheduling/retry (decisão #4 ADR confirmada).
6. Build limpo + tests unitários.
7. **Smoke test**: campanha de 5 contatos com 1 imagem + 1 texto, em sandbox uazapi. Anexar log de execução no PR.

## Dev Notes

### Refator do `getMessageOptions`
Antes (formato Baileys, retornado pra `wbot.sendMessage`):
```ts
return { image: fs.readFileSync(path), caption: body, mimetype };
```

Depois (formato uazapi):
```ts
return { type: "image", file: base64OrUrl, text: body, mimetype };
```

Onde `file` pode ser:
- URL HTTPS pública (preferido pra mídias servidas pelo nosso backend)
- Base64 inline (fallback pra mídias locais que não têm URL pública)

### Pseudocódigo loop de campanha
```ts
for (const shipping of campaignShippings) {
  try {
    const options = await getMessageOptions(file.path, ...);
    if (options.type === "text") {
      await SendText(whatsapp, { number: chatId, text: options.text });
    } else {
      await SendMedia(whatsapp, { number: chatId, ...options });
    }
  } catch (err) {
    if (err.code === "ERR_UAZAPI_RATE_LIMITED") {
      // re-enqueue com delay maior
      await campaignQueue.add(...);
      break;
    }
    throw err;
  }
  await delay(campaign.delay);
}
```
